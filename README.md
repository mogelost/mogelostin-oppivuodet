# Mögelostin oppivuodet

Esseekokoelma Möbius-nauhasta rakenteellisena käsitteenä, emergenssistä, abstraktiosta, tietoisuudesta, tekoälystä ja fysikaalisista analogioista. Tekstit ovat syntyneet keskustelujen pohjalta ja koottu yhdeksi kaareksi.

> Tämä aika rakastaa varmuutta. Siksi puheesta on tullut jäykkää –  
> lausumme toteamuksia, emme havaintoja. Mutta todellisuus ei ole näin suoraviivainen.  
>
> Kun ihminen oppii ajattelemaan Möbius-nauhana, hän alkaa puhua toisin.  
> Hän ei enää väitä, että “on näin”, vaan että “tässä kohdassa se näyttää tältä”.  
> Tuo pieni ero muuttaa kaiken.  
>
> Se ei tee kielestä epätarkkaa, vaan elävää. Se palauttaa liikkeen ajatteluun –  
> ja sen myötä mahdollisuuden ymmärtää toista.
