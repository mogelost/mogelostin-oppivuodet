# S01 – Liikkeelle!

Tämä kausi on käynnistys. Ei teoriaa ensin, vaan siirtymä: väitteestä havaintoon.

Tässä kohdassa kokoelma ottaa askeleen pois varmuudesta. Kieli alkaa liikkua: “on” muuttuu muotoon “tässä kohdassa näyttää”. Ajattelu ei pehmene – se tarkentuu. Kun väite lakkaa olemasta lukko, se muuttuu työkaluksi.

Jos myöhemmissä kausissa rakennetaan rakenteita (mieli–keho, aika, emergenssi, Möbius), tässä harjoitellaan ensin sitä taitoa, jolla niitä voi ylipäätään nähdä: keskeneräisyyden sietoa ilman että merkitys katoaa.

## Miten lukea tämä kausi

- Jos haluat *avaimen koko kokoelmaan*, aloita E01:stä.
- Jos haluat *kehystä ja väitteen*, E02 piirtää taustan (tiede–filosofia).
- Jos haluat *nopean koukun*, E03 vie arjesta suoraan tietoisuuden kysymyksiin.
