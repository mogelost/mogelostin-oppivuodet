# S02 – Ihmisen käyttöjärjestelmästä: keho, mieli, aika ja tahto

Tämä kausi siirtyy “sisään”. Ei mystiikan kautta, vaan järjestelmän kautta: keho ja mieli eivät ole kaksi eri asiaa, vaan sama ilmiö eri tarkkuustasoilla.

Tässä luvussa Möbius lakkaa olemasta pelkkä kuva ja alkaa toimia ajattelun työkaluna. Kehon signaalit muuttuvat tunteiksi, tunteet muuttuvat valinnoiksi, valinnat muuttuvat ajaksi ja ajassa virheestä tulee oppimisen rytmi. Vapaus ei näyttäydy irtautumisena, vaan kykynä suunnata liikettä tietoisesti — rajoissa, jotka tekevät suunnasta merkityksellisen.

Kausi päättyy kohtaan, jossa yksilö ei enää kanna kaikkea yksin: yhteys alkaa näyttää siltä, että se on rakenteellinen ehto, ei lisäosa.
